Dublin House Prices

The Dublin House Price 2016 Data is divided into five parts:

  1. 01_sourcing_the_housing_data, which deals with getting the data from the Property Price Register and using Google maps to add to it;
  2. 02_cleaning_the_dublin_house_price_data describes the different data munging we need to do to make the data ready for analysis;
  3. The initial data has 33% of the post code / eirKey data missing. 03_finding_missing_values_with_naive_bayes uses machine learning to fill in those missing values;
  4. 04_analysing_dublin_house_price_data does exactly what it says on the tin, and then
  5. 01_sourcing_the_housing_data draws some maps, because what is housing data without maps, after all?
